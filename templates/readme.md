# 提示词模版 (Templates Layer)

### 职责
存放所有 Agent 的“思维说明书”。通过 Prompt Engineering（提示词工程）实现解耦，方便在不修改代码的情况下快速调整 Llama-3 的表现。

### 核心文件
- `planner_cot.txt`: **思维链模版**。包含“强制物理思考”指令，要求模型在输出路线前先进行限重、限速、封路逻辑的预判。
- `narrator.txt`: **解说员模版**。负责将高德生成的干燥路径点转化为具有“情绪价值”和“安全提示”的自然语言解说。

### 优化策略
- 使用变量占位符（如 `{weather}`, `{constraints}`）与实时 API 数据动态拼接。
